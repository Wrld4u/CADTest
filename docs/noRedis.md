# Нагрузочный тест без Redis (`load-test-mixed`)

Дата: 2026-03-19

## Профиль

- Команда: `make load-test-mixed`
- Длительность: `20s`
- Смешанная нагрузка:
  - `160` соединений на unique-поток (`user_id`/`seat_id` разные)
  - `40` соединений на hotspot-поток (один и тот же `seat_id`)

## Краткая сводка

- Hotspot-поток (`40 connections`):
  - `1` успешная бронь (`2xx`), `486` неуспешных (`non-2xx`) — ожидаемо для одного `seat_id`
  - `527 requests in 20.05s`
  - `Avg latency: 1580.79 ms`, `p99: 2985 ms`
- Unique-поток (`160 connections`):
  - `2k requests in 20.28s, 703 kB read`
  - `Avg latency: 1788.04 ms`, `p99: 6016 ms`
  - `Req/Sec Avg: 86.2`
